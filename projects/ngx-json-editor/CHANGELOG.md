# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-15

### Added
- Initial release of ngx-json-editor
- JSON editor component with syntax highlighting
- Real-time JSON validation
- Toolbar with customizable buttons:
  - Format JSON with indentation
  - Minify JSON (remove whitespace)
  - Sort keys alphabetically
  - Search and highlight functionality
  - Copy JSON to clipboard
  - Download JSON as file
  - Reset to original data
  - Load JSON from file
- Standalone components support
- TypeScript support with full type definitions
- Configurable toolbar via `JsonEditorConfig`
- Separate `JsonEditorControlsComponent` for toolbar
- `JsonSearchComponent` for search functionality
- Angular 18+ compatibility

### Features
- Built with modern Angular standalone components
- Reactive and template-driven forms compatible
- Modular architecture with separate components
- Comprehensive documentation and examples
